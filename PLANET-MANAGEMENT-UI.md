# Planet Management UI Documentation

## Overview

The Planet Management Panel is a comprehensive React component for managing planets in the space strategy game. It provides an intuitive interface for viewing resources, constructing buildings, building ships, and managing construction queues with real-time updates.

## Resource System

### Resource Types

```typescript
interface Resources {
  minerals: number;  // Primary building resource
  energy: number;    // Powers buildings and ships
  crystal: number;   // Rare resource for advanced tech
```

### Production Rates

Resources are produced by buildings:

- **Mineral Mine** - Produces minerals per minute
- **Solar Power Plant** - Produces energy per minute
- **Habitat** - Increases population capacity

### Real-Time Updates

Resources update every second (60 times per minute):

```typescript
useEffect(() => {
  const interval = setInterval(() => {
    setResources(prev => ({
      minerals: prev.minerals + production.minerals / 60,

  return () => clearInterval(interval);
```

## Construction Queue

### Queue Management

- **Buildings Queue** - Buildings and upgrades
- **Ships Queue** - Ship construction
- **Independent Queues** - Can build ships while upgrading buildings

### Progress Tracking

Real-time progress bars update every 100ms:

```typescript
const progress = (Date.now() - startTime) / (completionTime - startTime) * 100;
```

### Cancellation

Cancel any queued item and get partial refund:

```typescript
onCancelConstruction(queueItemId);
```

## Performance

### Optimizations

1. **Memoization** - Components memoized with `React.memo`
2. **Virtual Scrolling** - For large building lists (optional)
3. **Debounced Updates** - Resource updates throttled
4. **Lazy Loading** - Tab content loaded on demand

### Benchmarks

- **Initial Render**: < 50ms
- **Resource Update**: < 5ms
- **Queue Update**: < 10ms
- **Memory Usage**: ~2MB

## Troubleshooting

### Resources Not Updating

- Check if `planet.buildings` is populated
- Verify production calculation
- Ensure interval is running

### Buttons Disabled

- Check resource availability
- Verify `canAfford` function
- Check building max level

### Queue Not Showing

- Verify `buildingQueue` and `shipQueue` props
- Check completion times
- Ensure queue items have correct format
