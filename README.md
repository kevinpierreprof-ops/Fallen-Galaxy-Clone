## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- A modern web browser (Chrome, Firefox, Safari, or Edge)

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/kevinpierreprof-ops/Fallen-Galaxy-Clone.git
   cd Fallen-Galaxy-Clone
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open your browser and navigate to `http://localhost:8080`

### Alternative: Open Directly

You can also open `index.html` directly in your web browser without running a server.

## How to Play

1. Click **Start Game** to begin
2. Watch your planets produce resources
3. Click on planets to select them
4. Manage your space empire and compete against opponents
