## Quick Start

### 1. Import and Use Hook

```typescript
import { useAudio } from '@/hooks/useAudio';
import { SoundEffectType } from '@/services/AudioManager';

function MyComponent() {
  const { playMusic, playSound, isReady } = useAudio();

  useEffect(() => {
    if (isReady) {
      playMusic('main-menu');
    }
  }, [isReady, playMusic]);

  return (
    <button onClick={() => playSound(SoundEffectType.CLICK)}>
      Click Me
    </button>
  );
}
```

### 2. Add Audio Control Panel

```typescript
import { AudioControlPanel } from '@/components/AudioControlPanel';

function Settings() {
  return (
    <div>
      <h1>Settings</h1>
      <AudioControlPanel />
    </div>
  );
}
```

## API Reference

### AudioManager

Singleton class managing all audio.

```typescript
const audioManager = AudioManager.getInstance();
```

#### Methods

**Initialization:**
- `initialize(): Promise<void>` - Initialize and preload audio
- `isReady(): boolean` - Check if initialized
- `getPreloadProgress(): number` - Get preload progress (0-100)

**Music:**
- `playMusic(trackId: string, fadeIn?: boolean): void` - Play music track
- `stopMusic(fadeOut?: boolean): void` - Stop current music
- `setPlaylist(trackIds: string[]): void` - Set music playlist
- `playNextInPlaylist(): void` - Play next track
- `playPreviousInPlaylist(): void` - Play previous track

**Sound Effects:**
- `playSoundEffect(effectType: SoundEffectType, sprite?: string): void` - Play sound effect

**Volume:**
- `setMasterVolume(volume: number): void` - Set master volume (0-1)
- `setMusicVolume(volume: number): void` - Set music volume (0-1)
- `setEffectsVolume(volume: number): void` - Set effects volume (0-1)

**Mute:**
- `toggleMusicMute(): void` - Toggle music mute
- `toggleEffectsMute(): void` - Toggle effects mute
- `muteAll(): void` - Mute all audio
- `unmuteAll(): void` - Unmute all audio

**Getters:**
- `getSettings(): AudioSettings` - Get current settings
- `getCurrentTrackInfo(): { id: string | null; playing: boolean }` - Get current track

**Cleanup:**
- `destroy(): void` - Cleanup and unload all audio

### Example 1: Play Music on Mount

```typescript
function MainMenu() {
  const { playMusic, isReady } = useAudio();

  useEffect(() => {
    if (isReady) {
      playMusic('main-menu', true); // Fade in
    }
  }, [isReady, playMusic]);

  return <div>Main Menu</div>;
}
```

## Performance

### Preloading

All audio files are preloaded on initialization:

```typescript
const { preloadProgress, isReady } = useAudio();

if (!isReady) {
  return <div>Loading audio... {preloadProgress}%</div>;
}
```

### Memory Management

- Audio files are cached in browser memory
- Use `destroy()` to unload all audio when done
- Singleton pattern ensures only one instance

### Optimization Tips

1. **Use OGG format** for better compression
2. **Compress audio files** to reduce size
3. **Limit simultaneous sounds** to avoid audio glitches
4. **Use audio sprites** for multiple short sounds in one file

## Troubleshooting

### Audio Not Playing

**Issue:** No sound when calling `playMusic()` or `playSound()`

**Solutions:**
1. Check if `isReady` is `true`
2. Verify audio files exist in `public/audio/`
3. Check browser console for errors
4. Ensure volume is not 0 or muted
5. Check browser autoplay policy (user interaction required)

### Crackling or Glitching

**Issue:** Audio crackles or glitches

**Solutions:**
1. Reduce number of simultaneous sounds
2. Lower audio quality/bitrate
3. Use audio sprites for short sounds
4. Check CPU usage (other processes)

### Volume Not Changing

**Issue:** Volume slider doesn't affect audio

**Solutions:**
1. Check if audio is muted
2. Verify localStorage is working
3. Check if master volume is 0
4. Ensure Howler.js is loaded

### Settings Not Persisting

**Issue:** Settings reset on page reload

**Solutions:**
1. Check browser localStorage is enabled
2. Verify no errors in console
3. Check localStorage quota
4. Clear localStorage and try again

## Best Practices

1. **Always check `isReady`** before playing audio
2. **Fade transitions** for better UX
3. **Provide mute options** for accessibility
4. **Compress audio files** for faster loading
5. **Use multiple formats** (MP3, OGG) for compatibility
6. **Limit simultaneous sounds** to 3-5
7. **Respect user preferences** (volume, mute)
8. **Show loading indicator** during preload
