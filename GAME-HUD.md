# Game HUD (Heads-Up Display) Documentation

## Overview

The Game HUD is a comprehensive overlay interface that provides players with real-time information and quick access to game features. It includes resource displays, minimap, game controls, alliance status, and action buttons.

### 2. ResourceBar

Displays resources with income and storage indicators.

```typescript
<ResourceBar
  resources={{
    minerals: 1250,
    energy: 850,
    credits: 5420,
    population: 15000,
    storage: {
      minerals: 10000,
      energy: 5000,
      credits: 50000
    }
  }}
  income={{
    minerals: 120,
    energy: 95,
    credits: 45
  }}
/>
```

**Features:**
- Shows current amount / max storage
- Income rate per hour
- Color-coded storage bars (green/orange/red)
- Number formatting (1K, 1M)

### 3. Minimap

Interactive galaxy overview with planet selection.

```typescript
<Minimap
  planets={planets}
  currentPlanetId="planet-1"
  onPlanetSelect={(id) => console.log('Selected:', id)}
  width={200}
  height={200}
/>
```

**Features:**
- Canvas-based rendering
- Grid background
- Color-coded planets
- Current planet highlight (yellow ring)
- Click to select planet

### 5. GameControls

Time display and game speed controls.

```typescript
<GameControls
  gameSpeed="1x"
  currentTime={Date.now()}
  onSpeedChange={(speed) => setGameSpeed(speed)}
/>
```

**Features:**
- Year and day display (e.g., "Year 2100, Day 45")
- Time display (HH:MM:SS)
- Speed buttons: Pause, 1x, 2x, 4x
- Active speed highlighted

### 6. AllianceStatus

Displays current alliance information.

```typescript
<AllianceStatus
  alliance={alliance}
  onViewDetails={() => navigate('/alliance')}
/>
```

### 7. NotificationBadge

Unread message count indicator.

```typescript
<NotificationBadge count={5} maxDisplay={99} />
```

**Features:**
- Shows count (e.g., "5")
- Max display "99+" for large numbers
- Pulsing animation
- Hidden when count is 0

## Resource System

### Resource Types

```typescript
interface ResourceData {
  minerals: number;    // Mining resource
  energy: number;      // Power resource
  credits: number;     // Currency
  population: number;  // Population count

  };

```

### Income Calculation

Resources increase based on income per hour:

```typescript
const incomePerSecond = income.minerals / 3600;
newMinerals = Math.min(storage.minerals, current + incomePerSecond);
```

### Storage Indicators

Color changes based on storage percentage:
- **Green** (< 70%) - Plenty of space
- **Orange** (70-90%) - Getting full
- **Red** (> 90%) - Nearly full

## State Management

### Resource Updates

```typescript
useEffect(() => {
  const interval = setInterval(() => {
    setResources(prev => ({
      minerals: prev.minerals + income.minerals / 3600,
      energy: prev.energy + income.energy / 3600,
      credits: prev.credits + income.credits / 3600
    }));
  }, 1000);

  return () => clearInterval(interval);
}, [income]);
```

### Game Tick

```typescript
useEffect(() => {
  if (gameSpeed === 'paused') return;

  const multiplier = speedMultipliers[gameSpeed];
  const interval = setInterval(() => {
    setCurrentTime(prev => prev + 1000 * multiplier);
  }, 1000);

  return () => clearInterval(interval);
}, [gameSpeed]);
```

  // Open planet details
  setPlanetDetailsId(planetId);
};

<GameHUD
  {...props}
  onPlanetSelect={handlePlanetSelect}
/>
