# Audio Manager Quick Reference

## Setup

```bash
npm install howler
npm install --save-dev @types/howler
```

## Basic Usage

```typescript
import { useAudio } from '@/hooks/useAudio';
import { SoundEffectType } from '@/services/AudioManager';

const { playMusic, playSound, isReady } = useAudio();

// Play music
playMusic('main-menu', true); // with fade

// Play sound effect
playSound(SoundEffectType.CLICK);
```

## Music Tracks

| ID | Use Case |
|----|----------|
| `main-menu` | Main menu |
| `exploration` | Normal gameplay |
| `combat` | Combat scenes |
| `peaceful` | Ambient background |
| `victory` | Victory screen |

```typescript
SoundEffectType.CLICK            // UI click
SoundEffectType.HOVER            // UI hover

SoundEffectType.COMBAT_HIT       // Hit in combat
SoundEffectType.COMBAT_DESTROY   // Destroyed
SoundEffectType.MESSAGE_RECEIVED // New message
SoundEffectType.NOTIFICATION     // Notification
SoundEffectType.ERROR            // Error
SoundEffectType.SUCCESS          // Success
```

## Volume Controls

```typescript
const { setMasterVolume, setMusicVolume, setEffectsVolume } = useAudio();

setMasterVolume(0.8);  // 80%
setMusicVolume(0.6);   // 60%
setEffectsVolume(0.7); // 70%
```

## Mute Controls

```typescript
const { toggleMusicMute, toggleEffectsMute, muteAll, unmuteAll } = useAudio();

toggleMusicMute();    // Toggle music mute
toggleEffectsMute();  // Toggle effects mute
muteAll();            // Mute everything
unmuteAll();          // Unmute everything
```

## Playlist

```typescript
const { setPlaylist, playMusic, nextTrack, previousTrack } = useAudio();

setPlaylist(['exploration', 'peaceful', 'main-menu']);
playMusic('exploration');
nextTrack();     // Play next
previousTrack(); // Play previous
```

## Check Ready State

```typescript
const { isReady, preloadProgress } = useAudio();

if (!isReady) {
  return <div>Loading... {preloadProgress}%</div>;
}
```

## Common Patterns

### Button with Click Sound

```typescript
const { playSound } = useAudio();

<button onClick={() => {
  playSound(SoundEffectType.CLICK);
  doAction();
}}>
  Click Me
</button>
```

### Play Music on Page Load

```typescript
useEffect(() => {
  if (isReady) {
    playMusic('main-menu', true);
  }
}, [isReady, playMusic]);
```

### Combat Music Switch

```typescript
const startCombat = () => {
  playMusic('combat', true);
  playSound(SoundEffectType.COMBAT_START);
};

const endCombat = () => {
  playMusic('exploration', true);
};
```

### Notification Sound

```typescript
useEffect(() => {
  if (newMessage) {
    playSound(SoundEffectType.MESSAGE_RECEIVED);
  }
}, [newMessage]);
```

```typescript
// Automatically saved to localStorage
{
  masterVolume: 0.8,
  musicVolume: 0.6,
  effectsVolume: 0.7,
  musicMuted: false,
  effectsMuted: false
}
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| No sound | Check `isReady`, verify files exist |
| Volume not working | Check mute state, master volume |
| Settings not saving | Check localStorage enabled |
| Audio glitching | Reduce simultaneous sounds |
